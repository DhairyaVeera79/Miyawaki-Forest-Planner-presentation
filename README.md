# Miyawaki Forest Planner Presentation Sub-Repo

This folder is a standalone, static presentation package for stakeholders.

## Contents
- `index.html` — single-page presentation
- `styles.css` — presentation styling
- `TEMP_MFP_PRESENTATION_HANDOFF.md` — handoff notes and requirements snapshot

## GitHub Pages
1. Push this folder as its own repository.
2. In repository settings, enable **Pages**.
3. Set source to **Deploy from a branch**.
4. Choose branch `main` and folder `/ (root)`.
5. Your site URL will be:
   - `https://<org-or-user>.github.io/<repo-name>/`

## Local Preview
Open `index.html` directly in a browser, or run a lightweight static server from this folder.
